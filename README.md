# Portfolio - Iker Hernández Santana

Este es tu portfolio personal construido con React, TypeScript, Tailwind CSS y Framer Motion. 
El diseño tiene una temática oscura, "brutalista" y utiliza un color rojo de acento muy saturado junto con animaciones y efectos tipo glitch.

## 🚀 Empezando

1. Instala las dependencias:
\`\`\`bash
npm install
\`\`\`

2. Inicia el servidor de desarrollo:
\`\`\`bash
npm run dev
\`\`\`

## 📝 Cómo personalizar tus datos

El sitio ha sido diseñado pensando en la facilidad de uso. Puedes cambiar todo tu contenido sin necesidad de entrar en los componentes de estilo complejo.

### 🖼️ Cambiar tu foto de perfil

1. Se utiliza una imagen en el componente \`src/components/Hero.tsx\`.
2. Busca la etiqueta \`<img src="..." />\` y cambia la URL o importar una imagen de tus archivos locales. 

### 💼 Modificar tu Experiencia y Proyectos

Toda la información del timeline y del portafolio reside en la carpeta \`src/data/\`.

- **Experiencia**: Edita \`src/data/experience.ts\`
- **Proyectos**: Edita \`src/data/projects.ts\`

### 🔗 Enlaces y Redes Sociales

- Para editar los enlaces del pie de página, dirígete a \`src/components/Footer.tsx\`.

## 🛠 Stack Tecnológico

- **React 18** + **Vite**
- **TypeScript**
- **Tailwind CSS v4** (todo el estilo está en \`src/index.css\` y clases en línea)
- **Framer Motion** (animaciones complejas y staggering)
- **Lucide React** (iconos SVG ligeros)

## 🎨 Tipografías y Colores

- **Fuentes**: Bebas Neue (títulos grandes) y Space Mono (cuerpos de texto y datos). 
- **Colores**: Negro sólido como fondo y Rojo brillante (\`#CC0000\`) como color principal que resalta las partes importantes de la web.
